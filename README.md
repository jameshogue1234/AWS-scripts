# AWS-acripts
All aws scripts.

```
Running `add_tags.py`
EXAMPLE:
python add_tags.py -k Name -v JamesH_test_okDelete -K HighScore -V jch

Look up an instance with a specific tagged key/value pair. Can be any pair.
k = key to lookup (part of k/v lookup)
v = value to lookup (part of k/v lookup)

This will take list created above and add a new tag
K = Key to be added to match above
V = Value to be added to match above
```
